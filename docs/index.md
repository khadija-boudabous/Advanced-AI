# Welcome

Welcome to the **Advanced AI** practical programming documentation.

This site contains materials, instructions, and references for each programming practical session in the Advanced AI course.

## Overview

Each practical session is organized as a self-contained project exploring a key topic in modern AI. You will find detailed instructions, code walkthroughs, and exercises in the **Programming Practicals** section.

## Getting Started

To set up the environment, install [uv](https://docs.astral.sh/uv/) and run:

```bash
uv venv
uv sync --group student
```

Then activate the virtual environment:

```bash
# macOS / Linux
source .venv/bin/activate

# Windows
.venv\Scripts\activate
```
